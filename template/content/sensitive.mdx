# Sensitive Material

This section demonstrates a **tier-2 "hidden" section** (`minTier: 2,
belowTier: "hidden"`).

A stage-1 recipient never sees this section at all — not in the index, not in
the nav, and `/room/sensitive` is indistinguishable from a page that doesn't
exist. Use this mode when advertising the section's *existence* would itself
plant an unwanted question: personnel matters in an M&A room, a disputed item
in an application, anything where "on request" invites the wrong conversation.

Only stage-2 recipients (and above) know this page is here.
